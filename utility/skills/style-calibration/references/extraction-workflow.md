# Style Extraction Workflow

This reference fully defines Extract mode. Use it to produce a structured, evidence-backed writing style profile from representative samples.

## Input requirements

- 3-10 texts, same author, preferably same genre.
- 5,000-20,000 characters total when possible.
- Label genre and audience: technical review, fiction, newsletter, academic prose, product memo, etc.
- Different genres require different extraction emphasis: poetry needs rhythm and line breaks; technical docs need term density and structure; essays need argument movement.
- Mark quoted, translated, dialogue-heavy, edited, or co-authored material as non-author signal.
- For cross-genre style DNA, use at least 2 genres with 2+ samples each.
- For Chinese-English mixed texts, distinguish deliberate bilingual style from unavoidable technical terms.

## Rules

1. Every conclusion must cite original text evidence.
2. Prioritize differentiating features over common traits.
3. Avoid vague adjectives; write executable rules.
4. Quantify before qualifying.
5. Mark sample limitations explicitly.
6. Separate author-stable features from genre-specific features.

## Step 1: Quantitative baseline

Calculate or estimate:

- sample count and total characters
- total sentences and paragraphs
- average sentence length and variance
- longest and shortest sentence
- long/short sentence ratio
- sentence length distribution: 0-10, 11-20, 21-30, 31-50, 50+ characters or words
- average paragraph length by sentences and characters
- function word frequency per 1000 characters or words
- punctuation frequency per 1000 characters or words
- pronoun ratio and point of view
- formalness and content word distribution

## Step 2: Surface language features

For each dimension, include at least one source-text example:

- Vocabulary: term types, modifier intensity, signature expressions, avoided words.
- Function words: top differentiating function words and usage patterns.
- Sentence patterns: declarative/question/imperative/rhetorical ratio, rhythm logic, comma chaining.
- Punctuation and formatting: special marks, paragraph splitting, list/table preference.
- Rhetoric: metaphor, analogy, irony, contrast, restraint, intensity.

## Step 3: Structure and narrative

Extract:

- opening patterns: hook, question, thesis-first, story, quote, personal anecdote, context setup;
- body logic: parallel, progressive, contrast, problem-solution, timeline, claim-evidence-reasoning;
- closing patterns: summary, call-to-action, inversion, unresolved tension, final quote;
- transition habits;
- information density;
- point of view and stance.

## Step 4: Deep tone and thinking patterns

Use only text-derived evidence. Do not speculate about private beliefs.

Capture:

- emotional baseline and intensity boundaries;
- default analytical framework;
- judgment criteria;
- preferred causal chains;
- personal signature traits;
- recurring cognitive moves.

## Step 5: Taboos and anti-rules

Absence can be diagnostic. Mark the evidence basis:

- never observed across samples;
- observed once then absent;
- present only in quoted or non-author material;
- genre-specific absence.

Taboos may include vocabulary, sentence types, punctuation, rhetorical moves, topics, stances, emotional posture, or formatting.

## Step 6: Example annotation

Provide:

- 2-3 positive representative excerpts with annotations;
- negative wrong-way examples for core features;
- correction direction for each negative example.

## Step 7: Self-check

Verify:

- every claim has original text evidence;
- each rule is executable by a writer;
- stability is labeled;
- common traits are not over-weighted;
- small-sample limitations are explicit.

## Step 8: External calibration

When possible:

- hold out one sample and validate profile predictions;
- test across genre if claiming style DNA;
- revise features that fail validation;
- mark unresolved uncertainty.
