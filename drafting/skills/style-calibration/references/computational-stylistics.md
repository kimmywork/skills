# Computational Stylistics

Use this reference to keep style extraction evidence-based instead of impressionistic.

## Core principles

1. Unconscious habits are more reliable than conscious choices.
2. Function words are style DNA: articles, prepositions, conjunctions, pronouns, auxiliaries, particles.
3. Punctuation density captures breathing rhythm and is a strong style signal.
4. Quantify before qualifying; numbers anchor interpretation.
5. Extract feature families separately: function words, punctuation, sentence patterns, rhetoric, content.
6. Prioritize differentiating features over common traits.
7. Style replication uses stylometry as evidence, but turns evidence into executable writing rules.

## Why function words matter

Function words carry grammar more than topic. They are difficult to consciously control and often stay stable across topic changes. Treat them as the highest-priority signal when samples are sufficient.

Rule of thumb from stylometry practice:
- function word habits are often stable across genres;
- content words vary heavily with topic;
- punctuation and paragraphing often reveal rhythm better than vocabulary.

## Stability hierarchy

| Feature | Stability | Cross-genre? | Use |
|---|---:|---|---|
| Function words | 5/5 | yes | style DNA and high-confidence rules |
| Punctuation density | 4/5 | mostly | rhythm and formatting behavior |
| Sentence length | 3/5 | partial | genre-sensitive rhythm |
| Structure patterns | 3/5 | partial | openings, transitions, closings |
| Rhetoric habits | 2/5 | usually no | genre-specific style |
| Content preferences | 1/5 | no | do not overgeneralize |

## Feature extraction order

1. Count function words.
2. Count punctuation and paragraph breaks.
3. Measure sentence and paragraph length.
4. Identify structure and transition patterns.
5. Identify rhetoric and tone.
6. Only then summarize content preferences.

## Interpretation rules

- If quantitative and qualitative impressions conflict, trust the numbers until the conflict is explained.
- If a feature appears in one sample only, mark it sporadic.
- If a feature appears in one genre only, mark it genre-specific.
- If a feature appears across genres, consider it style DNA.
- If samples are small, limit conclusions to stable quantifiable features and high-confidence taboos.
